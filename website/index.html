Welcome to B4wbem

B4wbem intends to be an administration system for Linux. It is based on libCIM, which serves as a basis for an implementation of the Common Information Model (CIM) which is part of the Web-Based Enterprise Management (WBEM) standards published by the Distributed Management Taskforce (DMTF). In particular, B4wbem offers a CIM Object Manager (CIMOM), and a sample CIM Client.
